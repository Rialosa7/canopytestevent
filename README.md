# 🤖 SmartGuard AI

A lightweight AI-powered application that helps users achieve their goals and stay safe online.

---

## 🚀 Features

### 🎯 AI Task Generator
- Analyze user goals using intelligent keyword detection
- Generate actionable and structured steps instantly
- Works with flexible natural language input

### 🔍 Scam Detection System
- Scans URLs for potential scam indicators
- Provides risk levels:
  - ✅ Low Risk
  - ⚠️ Medium Risk
  - 🚨 High Risk
- Explains reasons behind the detection

### 💡 AI Productivity Tips
- Generates random motivational tips
- Enhances user experience with dynamic output

---

## 🧠 How It Works

SmartGuard AI uses lightweight AI logic (rule-based intelligence) to:
- Understand user intent from text input
- Match patterns and keywords
- Generate dynamic and useful outputs
